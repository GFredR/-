# Graduate_live<br>
基于iOS的视频直播App<br>
毕业设计的项目，关于视频直播<br>
主要功能：第三方登录、加载首页内容、观看直播、主播进行直播、定位。<br>
架构模式：MVVM。<br>
主要框架：Alamfire、Kingfisher、ijkplayer、LFLiveKit、友盟集成第三方登录框架。<br>
语言：Swift4.2<br>
主要技术：第三方登录、音视频采集、音视频编码解码、推流、拉流、获取当前经纬度、加载网络图片。<br>
